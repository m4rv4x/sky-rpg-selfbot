# BETA Self Bot to play Sky RPG Game

## quick install
```bash
git clone https://github.com/m4rv4x/skyrpgbot/
cd skyrpgbot
npm install
```
## add your token in config
```bash
cat config.yaml
```
## run it
```bash
node .
```
